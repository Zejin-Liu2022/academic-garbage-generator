# 快速上手指南

## 🚀 5分钟快速开始

### 第一步：打开应用
```bash
# 方法1: 双击文件
打开 src/editor.html

# 方法2: 使用命令行（macOS）
open src/editor.html

# 方法3: 使用命令行（Windows）
start src/editor.html

# 方法4: 使用命令行（Linux）
xdg-open src/editor.html
```

### 第二步：配置 API（首次使用）
1. 点击右上角 ⚙️ 设置图标
2. 在"API 配置"中填写：
   - **API Key**: 你的 OpenAI API Key
   - **Base URL**: `https://api.openai.com/v1`（或其他兼容服务）
   - **Model Name**: `gpt-3.5-turbo`（或其他模型）
3. 设置会自动保存

### 第三步：创建第一个文档
1. 点击左上角 ☰ 菜单
2. 点击 **[+ 新建写作]** 按钮
3. 在文档标题处输入标题
4. 在编辑区域开始写作或使用 AI 生成

## 📁 文件夹管理

### 创建文件夹
```
1. 打开左侧边栏（☰ 菜单）
2. 找到"文件夹"区域
3. 点击右侧的 [+] 图标
4. 输入文件夹名称
5. 点击确定
```

### 移动文档到文件夹
```
1. 在文档列表中找到要移动的文档
2. 将鼠标悬停在文档上
3. 点击文件夹图标 📁
4. 从弹出菜单中选择目标文件夹
```

### 查看文件夹内容
```
1. 点击文件夹名称
2. 文档列表只显示该文件夹内的文档
3. 点击"全部文档"显示所有文档
```

## 🔍 搜索文档

### 基础搜索
```
1. 打开左侧边栏
2. 在搜索框中输入关键词
3. 文档列表实时过滤
4. 清空搜索框显示所有文档
```

### 高级搜索（组合使用）
```
场景：在"课程论文"文件夹中搜索包含"AI"的文档

步骤：
1. 点击"课程论文"文件夹
2. 在搜索框输入"AI"
3. 只显示该文件夹内匹配的文档
```

## 🔀 排序文档

### 更改排序方式
```
1. 打开左侧边栏
2. 点击排序下拉菜单
3. 选择排序方式：
   - 按修改时间（最近编辑的在前）
   - 按创建时间（最新创建的在前）
   - 按标题（A-Z 字母顺序）
```

## ✍️ AI 辅助写作

### 生成提纲
```
1. 在左侧设置区选择"思路引导（生成提纲）"
2. 输入主题，例如："人工智能在教育中的应用"
3. 点击"开始生成"
4. AI 将生成结构化的提纲
```

### 快速成文
```
1. 选择"快速成文（生成内容）"
2. 输入主题和要求
3. 设置字数（例如：2000）
4. 选择引用规范（GB/T 7714、APA、MLA）
5. 点击"开始生成"
```

### 内容润色
```
1. 选择已有内容
2. 选择"内容润色（修改扩写）"
3. 输入具体要求，例如："使这段内容更加学术化"
4. 点击"开始生成"
```

## 💾 保存和导出

### 自动保存
- 应用会自动保存你的编辑
- 无需手动点击保存按钮
- 数据存储在浏览器的 localStorage

### 导出为 Word
```
1. 点击顶部"导出 Word"按钮
2. 文档自动下载为 .docx 文件
3. 可在 Microsoft Word 或 WPS 中打开
```

### 导出为 PDF
```
1. 点击"导出 PDF"按钮
2. 文档自动生成 PDF
3. 可在任何 PDF 阅读器中查看
```

## 🎨 界面自定义

### 切换主题
```
1. 点击右上角 ⚙️ 设置
2. 在"界面外观"中选择：
   - 跟随系统（自动）
   - 浅色模式
   - 深色模式
```

### 调整缩放
```
方法1: 使用滑块
1. 在格式工具栏右侧找到缩放控制
2. 拖动滑块调整（60%-500%）

方法2: 输入数值
1. 在缩放控制的输入框中输入数字
2. 按回车确认

方法3: 滚轮缩放
1. 按住 Ctrl/Cmd 键
2. 滚动鼠标滚轮
```

## 🛠️ 常见问题

### Q: 我的数据存在哪里？
A: 数据存储在浏览器的 localStorage 中，仅在你的设备上。

### Q: 清除浏览器缓存会删除数据吗？
A: 是的，清除缓存会删除所有文档。请定期导出重要文档。

### Q: 可以在多个浏览器间同步吗？
A: 目前不支持。计划在未来版本中添加云同步功能。

### Q: 支持移动设备吗？
A: 建议在桌面浏览器使用。移动设备可能体验欠佳。

### Q: 文档数量有限制吗？
A: localStorage 约 5-10MB 限制，建议不超过 500 篇文档。

### Q: 如何备份数据？
A: 目前通过导出单个文档。未来将支持批量导出/导入。
